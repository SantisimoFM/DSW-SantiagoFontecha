Etiqutas basicas de HTML

las Etiquetas de HTML son una serie de codigos que se utilizan para definir los elementos que componen una pagina web, estos elementos pueden ser texto, imagines, videos, etc. las etiquetas son interpretadas por los programs conocidos como navegadores o exploradores web (por ejemplo Google Chrome, Microsoft Edge, opera, etc) quienes se encargan de mostrar al usuario la pagina web de manera adecuada para su interaccion. exepto algunas exepciones las mayoria de las etiquetas estan formadas por 2 tipos de etiqueta una de apertura y otra de cierre, compuestas de la siguiente forma

  • etiqueta de apertura = caracter <, seguido del nombre de la etiqueta sin espacio es blanco y terminando con el caracter >.
  • etiqueta de cierre = caracter <, seguido del caracter /, seguido del nombre de la etiqueta sin espacio es blanco y terminando con el caracter >.

etiquetas basicas

  • "b" utilizala para aplicar negrilla
  • "i" utilizala para aplicar cursiva
  • "u" utilizala para aplicar subrayado
  • "font" utilizala para dar fotmato al texto
  • "h1" a "h6" utilizala para titulos y o subtitulos
  • "br" utilizala para hacer un salto de linea en un parrafo
  • "p" utilizala para crear un parrafo
  • "hr" utilizala para insertar una linea divisoria
  • "ul" utilizala para iniciar una lista no numerada
  • "ol" utilizala para iniciar una lista numerada
  • "li" utilizala dentro de ol o ul para agregar objetos a la lista
  • "img" utilizala para agregar una imagen
  • "table" utilizala para agregar una tabla
  • "tr" utilizala para agregar columnas a una tabla
  • "td" utilizala para agregar filas o celdas a una tabla
  • "a" utilizala para agregar enlaces
